# NinjaGirl-game

c# game using system windows form and drawing.
This is a simple game made for a multimedia course.

to test the game:
  - move with arrows
  - jump with space
  - to throw multiple kunai press m
  - to throw a single kunai at a time press n
  - to use laser press L
